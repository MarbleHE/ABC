\doxysubsection{Detailed Description}
An \mbox{\hyperlink{class_expression_list}{Expression\+List}} is an order list of (potentially null) expressions. However, more specifically, it\textquotesingle{}s intended to be a rough equivalent to the \{a,b,c\} notation in C++ Note that this is NOT a target (lvalue) but instead an rvalue So we could assign an \mbox{\hyperlink{class_expression_list}{Expression\+List}} into a \mbox{\hyperlink{class_variable}{Variable}} using a Variable\+Assignment And this would turn the \mbox{\hyperlink{class_variable}{Variable}} into a vector-\/valued \mbox{\hyperlink{class_variable}{Variable}} However, we cannot assign something to an \mbox{\hyperlink{class_expression_list}{Expression\+List}}
