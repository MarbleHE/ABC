\doxysubsection{Detailed Description}
A custom exception class to be used to break out of the Runtime\+Visitor in case that we visited a \mbox{\hyperlink{class_return}{Return}} statement. This is important for programs that use \mbox{\hyperlink{class_return}{Return}} statements to prematurely exit a program, e.\+g., in the body of a \mbox{\hyperlink{class_for}{For}} loop. The exception must be caught (and ignored) by the caller.
