Detailed Description

template<typename T>
class EmptyIteratorImpl< T >

EmptyIteratorImpl is a "dummy" iterator that can be used for node classes that do not have children Any two EmptyIteratorImpl for the same node will always be equal, i.e begin() == end() if used for begin/end
